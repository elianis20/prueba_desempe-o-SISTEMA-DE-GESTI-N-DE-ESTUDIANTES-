 DESCRIPTION

CREATE A STUDENT MANAGEMENT SYSTEM AND FACILITATE INFORMATION COLLECTION

THE OBJECTIVE WAS TO CREATE A SYSTEM THAT FACILITATES STUDENT INFORMATION REGISTRATION
WHERE TO VISUALIZE THE FOLLOWING DATA: REGISTRATIONS, QUERIES, UPDATES, DELETIONS, AND STORAGE OF INFORMATION

FUNCTIONS

The system allows:
- Adding new students
- Consulting the student list
- Updating information
- Deleting information
- Exiting the menu

The program's main structure is the use of dictionaries to store information

FUNCTIONS USED

- STUDENT_REGISTRATION
ALLOWS US TO ADD NEW STUDENTS STUDENTS
- SEARCH_STUDENTS
Allows us to search for students in the system.
- VIEW_STUDENTS
Allows us to view the students on the list.
- DELETE_STUDENTS
Allows us to delete students who will no longer be part of the system.
- UPDATE_STUDENTS
Allows us to update student information.
